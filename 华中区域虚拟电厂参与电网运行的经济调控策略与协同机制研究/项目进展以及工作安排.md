# 研究内容
**1.** **华中区域虚拟电厂发展现状及资源灵活调控潜力分析**
调研华中区域虚拟电厂建设运营情况，包括数量、规模、类型等；分析华中区域虚拟电厂可调节资源特性，如分布式新能源、可控负荷、储能等；评估虚拟电厂参与省间电力市场的技术可行性和经济潜力。

**2.** **虚拟电厂参与电力系统调峰调频服务的调控策略及经济性分析**
从电网安全运行、电力平衡、系统调控等维度，选择典型场景研究虚拟电厂参与电力系统调峰调频服务的调控策略，包括设计实时功率分配策略，保证快速响应调频信号；设计实时调峰控制策略，协调虚拟电厂所含各类资源参与正反调峰等。同时从成本、收益两个维度对上述场景中的各种调控策略进行经济性分析。

**3.** **虚拟电厂参与省间电力现货市场机制研究**
完善虚拟电厂市场准入和退出机制；优化虚拟电厂参与省间（区域）日前、日内、实时等各时间尺度现货市场的交易规则，包括报价、出清、结算等；建立虚拟电厂提供辅助服务的补偿机制；完善虚拟电厂相关的技术标准和管理规范。

**4.** **虚拟电厂对电网运行的风险评估及应对管理策略研究**
对虚拟电厂成规模后市场力影响进行研究分析，评估对调控运行带来的风险；考虑不确定性的虚拟电厂运行风险应对方法，如预测误差、价格波动等，制定相应控制策略和管理机制。

# 中期检查结果
1. 研究内容一：
	1. 需将数据的时间维度进行全面更新，各项数据的截止时间应为2025年，不确定的数据可以先再列成调研需求；
	2. 查找退役机组改造为虚拟电厂的案例；
	3. 经济性评估的时候可以获取一些典型日细时间粒度发用电曲线来分析四省之间的调度潜力；
	4. 虚拟电厂作为爬坡资源的应用可行性；
	5. 对比不同省份虚拟电厂的分类以及不同类别虚拟电厂聚合什么样的资源；
2. 研究内容二：
	1. 当前虚拟电厂更多的作用体现在其调节功能上，并且在现货市场正式运行后，调峰市场将与之融合，建议分成现货正式运行前后，在语言描述方面进行相应调整，模型可以保留；
	2. 调频可以加入研究虚拟电厂中聚合什么样的资源可以参与调频市场以及如何调频（例如分布式储能）、性能如何评估等；
3. 研究内容三：
	1. 思考省内现货市场调研的必要性即为什么研究这部分内容；
	2. 可以从参与市场的种类（日前、日内等）、参与现货的交易方式等入手研究；
	3. 分析结算机制和偏差考核机制和其他运营主体的不同；
4. 研究内容四：
	1. 加入研究大规模并网后、频率大幅度波动、自身故障等对电网运行的风险防空，以及惯量控制等
5. 后续思考项目落地转化应用的方案
6. 思考资源多层级、多主体、多维度、多市场类型之间如何衔接
