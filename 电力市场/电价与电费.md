# 电价
我们常说的节点电价、统一结算价、电网代理费用这些都是电能量的费用，其他政府附加或者输配电费这些，都是不包含在内的，都是电网额外收取的，在电费单的后面呈现

**售电公司电费构成明细清单**

|费用类别|征收主体|核算标准|承担方|核心说明|
|---|---|---|---|---|
|电能量费用|发电企业|市场化交易价格（售电公司与发电企业协商确定）或基准电价（政策规定）|最终用电户|售电公司 “买电” 的核心成本，直接反映电能本身的价格|
|输配电价|电网企业|政府价格主管部门核定（按电压等级、用电类别分类定价）|最终用电户|覆盖电网建设、运维、损耗等成本，分为输电价和配电价两部分|
|政府性基金及附加|电网企业（代收代缴）|按政策固定标准征收（如可再生能源电价附加每千瓦时 0.019 元）|最终用电户|包括可再生能源电价附加、大中型水库移民后期扶持基金等，全额上缴政府|
|辅助服务费用|电网企业或电力交易中心|按电力市场规则核算（如调频、备用服务价格）|最终用电户|保障电力系统安全稳定运行的额外费用，部分纳入电能量费用或单独收取|
# 输配电费
| 对比维度   | 容量型输配电费                   | 电量型输配电费                |
| ------ | ------------------------- | ---------------------- |
| 计费核心依据 | 契约容量（kVA）或最大需量（kW）        | 实际用电量（kWh）             |
| 费用属性   | 覆盖电网固定成本（线路、变电站建设等投资）     | 覆盖电网变动成本（运维、电力损耗等）     |
| 计费方式   | 按每月固定单价 × 核定容量计算，金额相对稳定   | 按单价 × 实际用电量计算，随用电量波动   |
| 适用场景   | 新能源配套储能、大工业用户、占用电网容量资源的项目 | 普通工商业用户、居民用户、多数常规用电主体  |
| 缴纳逻辑   | 只要占用电网容量，无论是否满负荷用电都需缴纳    | 用电越多，缴纳费用越高，无用电则无此项费用  |
| 定价规则   | 政府核定分类单价（按电压等级、用电类别区分）    | 政府核定分类单价（按电压等级、用电类别区分） |
|        |                           |                        |
## 容量型输配电费产生原因
新能源配套储能等主体占用电网容量，核心是其接入和运行过程中，会长期占用电网的 “物理承载资源” 和 “调度预留资源”，哪怕实际没满负荷用电，电网也需为其预留足够容量以保障安全稳定，具体原因如下：
### 1. 接入电网需占用物理容量资源

- 储能项目（尤其是新能源配套储能）需通过特定电压等级的线路、开关设备接入电网，这些设备的额定容量是固定的。
- 一旦接入，该部分设备容量就被该储能项目 “占用”，无法再同时分配给其他用户或项目，形成物理层面的容量占用。

### 2. 运行特性要求电网预留调度容量

- 储能充电时属于用电负荷，放电时属于电源，其充放电功率存在峰值需求（如 100MW 储能的最大充电功率可达 100MW）。
- 电网需提前为其峰值功率预留对应的调度容量，避免多个项目同时满负荷充放电时，超出电网承载能力导致故障，这部分预留容量也属于被占用的电网资源。

### 3. 容量占用与实际用电量无关

- 哪怕储能项目实际充电量少、未达到峰值功率，其接入时核定的 “契约容量” 已明确了电网需为其预留的资源规模。
- 电网的固定成本（建设、设备折旧等）与容量占用直接相关，而非用电量，因此需通过容量型输配电费覆盖这部分成本。
