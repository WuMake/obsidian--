# 偏差考核 VS 偏差结算
1.  **偏差结算（经济结算）**
    *   **是什么**：这是对**实际发生的、与合同不符的电量**进行的一种**经济上的“多退少补”**。它的目的是公平地结算所有实际发出的电能。
    *   **怎么算**：无论你是发电方还是用电方，如果你实际发电/用电量（上网电量/用电量）与合同电量不一致，那么：
        *   **合同内电量**：按合同约定的价格结算。
        *   **偏差电量**：按市场上一个公允的参考价格（如绿电市场均价、全市场均价、现货价格）进行结算。
    *   **性质**：这是**常规的、非惩罚性**的结算行为，是电力交易中的正常环节。

2.  **偏差考核（惩罚性考核）**
    *   **是什么**：这是对**偏差过大**的行为进行的一种**经济惩罚**。它的目的是督促市场成员（特别是发电方）提高预测准确性，保障电网平衡和安全。
    *   **怎么算**：只有当你的偏差电量**超出了允许的宽容范围**（比如条款中提到的±30%），超出的部分才会被罚款。
    *   **性质**：这是**惩罚性**的，旨在约束不负责任的、过大的偏差行为。

**简单比喻**：
*   **偏差结算**：就像你买了10个苹果（合同），但果园实际摘了12个给你。你为10个苹果付合同价，多出的2个按市场价结算。这是公平交易。
*   **偏差考核**：就像你和果园约定好每天稳定供应10个苹果以保障你的商店运营，但果园连续多天只给你5个或给了你15个，导致你的商店经营困难或浪费严重。这时，你就会对果园这种不稳定的供应行为进行罚款。
### 日清分和月清分企业的不同

“清分”指的是对电量进行计量、确认和分割结算的过程。日清分和月清分的区别在于这个过程的频率和精细度。

| 特点 | **日清分企业** | **月清分企业** |
| :--- | :--- | :--- |
| **清分频率** | **每日** | **每月** |
| **数据基础** | 基于**每日**精确的计量数据 | 基于**整月**累计的计量数据 |
| **技术门槛** | **高**。需要具备高精度的计量设备和数据传输系统，能支持每日的数据采集和核算。 | **相对较低**。对计量和数据传输的实时性要求不高。 |
| **管理要求** | **高**。需要具备相应的管理能力，能够处理每日的海量数据和频繁的结算对账。 | **相对较低**。 |
| **为何在偏差考核上区别对待？** | 因为能够做到日清分，说明其**预测精度高、可控性强、与电网的互动性好**。在现货市场未运行时，电网调度已经可以相对精准地掌握其每日出力情况，对电网的冲击较小。因此，**暂不进行偏差考核**作为一种鼓励。 | 因为只能做到月清分，说明其**预测精度和可控性相对较差**，电网难以掌握其每日的精确波动，较大的偏差会对电网的日平衡造成冲击。因此，为了督促其提升管理水平，**继续进行偏差考核**。 |

### 总结

1.  **偏差结算（经济性）**：永远存在。是对所有实际电量的公平计价。
2.  **偏差考核（惩罚性）**：有条件存在。只针对偏差过大且管理能力较弱的市场主体。
3.  **规则逻辑**：
    *   **现货市场运行期间**：市场本身的高频价格信号（可能极高或极低）已经能很好地惩罚预测不准的行为了，所以**取消**额外的偏差考核。
    *   **现货市场未运行期间**：
        *   对**日清分**的“优等生”（技术和管理能力强），**暂不考核**，以示鼓励。
        *   对**月清分**的“普通生”，**继续考核**，设定±30%的宽容范围，超出部分按煤电基准价10%罚款，以督促其进步。
