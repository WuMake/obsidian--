# 电力交易机构
#电力交易机构
**定义**：电力交易机构是为各类市场主体提供规范、公开、透明的电力交易服务，负责市场交易组织、结算并根据需要履行交易合同管理职能的**非营利性公司制法人企业**。

**核心职能**：
1. **市场构建与注册管理**：
    
    - 负责搭建电力交易平台（包括线上交易系统）。
        
    - 受理市场主体的注册业务，管理其市场准入资格。
        
2. **交易组织**：
    
    - 根据市场规则，组织各类电力交易，包括**中长期交易**（年度、月度、多日等）、**现货交易**（日前、日内）以及**合同转让交易**等。
        
    - 发布交易公告、接收市场主体的报量报价信息。
        
3. **合同管理**：
    
    - 对达成交易的电力交易合同进行备案、管理。
        
    - 提供合同范本，并负责交易合同的归档、跟踪与执行监督。
        
4. **计划编制与提交**：
    
    - 将汇总形成的交易结果（如中长期交易合同、现货市场初步出清结果）编制成**交易计划**，并**提交给电力调度机构进行安全校核**。
        
5. **结算与清算**：
    
    - 根据交易合同和调度机构提供的**实际执行电量**等数据，进行电费计算、核对和出具结算依据。
        
    - 负责市场内的资金清算。
        
6. **信息发布**：
    
    - 向社会和市场主体发布市场交易信息、成交结果、结算信息等，确保市场公开透明。

# 电力调度机构
#电力调度机构 
**定义**：电力调度机构是依法依规设立，负责组织、指挥、指导和协调电网运行，保障电网安全、优质、经济运行，并**执行电力市场出清结果**的机构。

**核心职能**：

1. **电网安全分析与安全校核**：
    
    - 这是与交易机构协同的**关键环节**。调度机构接收交易机构提交的交易计划，利用电网模型进行**安全校核**，分析交易计划是否会引发电网阻塞或越限等安全问题，并将校核结果返回给交易机构。
        
2. **实时平衡与调度执行**：
    
    - 负责电网的**实时平衡**，确保任意时刻的发电与用电负荷相等。
        
    - 根据市场出清结果或安全校核后的交易计划，向发电厂、用户等下发布局精确的**调度指令**，控制发电和用电功率。
        
3. **组织辅助服务**：
    
    - 负责调用**调频、备用、无功调节、黑启动**等辅助服务资源，以维护电网的电能质量和安全稳定运行。在辅助服务市场中，调度机构通常是服务的购买者和调用者。
        
4. **现货市场出清**：
    
    - 在集中式现货市场中，调度机构通常是**出清计算的执行方**。它基于市场报价和电网实时安全约束，通过优化算法，计算出不同时段、不同地点的节点电价或区域电价，并形成最终的发电计划和调度指令。
        
5. **电网运行控制与事故处理**：
    
    - 负责电网的日常监视、操作和事故处理，在紧急情况下有权采取必要的干预措施，包括中止或调整市场交易结果，以优先保障电网安全。

## 电力交易机构与调度机构的协同

| **维度**   | **电力交易机构**                                                                                           | **电力调度机构**                                                                                          | **协同关系**                                                                                                                                                                |
| -------- | ---------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **核心职能** | **市场运营**：负责市场注册、交易组织、合同管理、信息发布、结算等[](https://nfj.nea.gov.cn/xwzx/jgdt/202501/t20250123_275693.html)。 | **电网物理运行**：负责电网安全稳定运行、实时平衡、执行交易结果[](https://nfj.nea.gov.cn/xwzx/jgdt/202501/t20250123_275693.html)。 | **功能互补**：一个管"电量"（经济流），一个管"电力"（物理流）。                                                                                                                                     |
| **角色定位** | **市场中台**：搭建交易平台，服务市场主体。                                                                              | **运行中枢**：保障电网安全，执行调度指令。                                                                             | 共同构成电力市场运行的"双引擎"。                                                                                                                                                       |
| **协作原则** |                                                                                                      |                                                                                                     | **市场调节优先，安全可靠兜底**[](https://zjic.zj.gov.cn/ywdh/nyhj/202510/t20251014_23742346.shtml)[](http://paper.people.com.cn/rmrbhwb/pc/content/202510/14/content_30109093.html)。 |

1. **交易组织阶段：交易在前，校核在后**
    - **流程**：交易机构首先根据市场规则组织交易，收集市场主体的报价、量等信息[](https://www.gov.cn/lianbo/bumen/202507/content_7031690.htm)。之后，将这些交易意向**统一提交给调度机构进行安全校核
    - **信息交流**：
        - 交易机构 → 调度机构：传递**交易申报信息**（如报量报价）、**初步交易结果**、**计划交易路径**等。
        - 调度机构 → 交易机构：返回**安全校核结论**，明确哪些交易可以执行，哪些因电网安全约束需要调整或终止
            
2. **市场出清与计划执行阶段**
    - 在现货市场中，调度机构会基于通过安全校核的交易情况，综合考虑电网实时运行状态，进行**市场出清**，形成最终的发电计划和调度指令。
    - 在**跨省跨区应急调度**等特殊场景下，调度机构会参考电力现货市场的报价、报量等信息来组织实施[](http://research.cnesa.org/report/info/?id=6453)。
        
3. **结算与事后阶段**
        - 交易机构根据最终实际执行的交易和调度结果，进行电费结算。同时，两大机构也需要共同或各自向监管部门和市场主体进行**信息披露**，保障市场公开透明[](https://www.gov.cn/zhengce/zhengceku/202507/content_7031714.htm)[](https://nfj.nea.gov.cn/xwzx/jgdt/202501/t20250123_275693.html)。

<mark style="background: #FFB86CA6;">涉及的信息：</mark>
- **交易信息**：市场主体的**注册资质**（交易机构提供）、**报量报价数据**、**交易合同**、**出清结果**等。
- **电网物理信息**：**网络拓扑结构**、**设备传输极限**、**安全约束条件**、**实时运行状态**、**负荷预测**等（主要由调度机构提供和判断）。
- **安全校核信息**：交易机构提交的**交易计划**，以及调度机构返回的**校核意见与阻塞管理结果。
- **执行与结算信息**：**最终的调度指令**、**实际执行电量**（用于结算）、**辅助服务调用情况**等。
- **跨电网交易信息**：在国家电网和南方电网之间，两大机构的信息交互更为复杂，需要通过交易平台实现**信息交互、供需匹配和结果共享。

## 自计划机组
自计划机组是电力市场中允许机组自主制定发电计划的类型，主要分为‌**完全自计划机组**‌和‌**部分接受调度的自计划机组**‌。其核心特征是机组可自主决定发电时段或容量，但需满足市场规则约束。

**定义与分类**
- 完全自计划机组：机组在市场中完全自主制定发电计划，不接受调度指令，以固定出力形式参与市场交易。
- 部分接受调度的自计划机组：机组在部分时段自主制定计划，其他时段需接受调度指令调整出力。 ‌

**运行机制**
- 自计划容量或时段：机组需提前申报自计划容量或时段，市场根据申报情况分配发电权。
- 与调度指令的关系：完全自计划机组不受调度指令约束，部分自计划机组在调度指令冲突时需调整计划。 ‌

**应用场景**
- 新能源机组：部分新能源机组（如风电、光伏）因出力波动性，可能采用自计划模式参与市场。
- 传统机组：部分火电或水电机组也可通过自计划模式优化运行效益。 ‌

**市场影响**
- 优势：提升机组灵活性，降低调度协调成本。
- 挑战：需平衡市场公平性与机组自主性，避免过度依赖自计划导致系统稳定性风险。 ‌

==对售电公司影响：==
1. 利润空间显著压缩
	以陕西为例，政策实施后售电公司批零价差被限制在0.012元/千瓦时以内，超出部分80%需返还用户。假设某公司月售电量1亿千瓦时、批零价差0.02元/千瓦时，每月需返还用户64万元仅保留16万元收益。
2. 行业分化格局加剧
	2024年上半年广东数据显示，独立售电公司度电收益为0.254元，发电企业背景公司为0.117元，电网背景公司为0.171元。超额收益分享机制对高收益独立售电公司形成更大经营压力。
3. 运营模式被迫转型
	政策倒逼企业从"价差套利"转向"服务增值"。如某央企虚拟电厂聚合17家用户，通过需求响应29次、调度负荷3910MW，累计收益达3300万元，探索出技术驱动型盈利路径。

==对市场的影响：==
1. 价格传导机制更趋完善
	通过组合类电价条款(如四川中长期交易均价90%+现货价格10%)、分时电价调整等措施，实现批发市场价格信号向零售端有效传导，促进用户优化用电行为响应市场波动。
2. 市场规范化程度显著提高
	政策倒逼售电公司提升运营效率，减少信息不对称。2025年浙江233家售电公司中，22家未提交自查材料3家未报送审计报告、26家不满足准入条件被整治，市场环境持续净化，
3. 市场参与主体行为转变
	用户从被动接受电价转向主动选择套餐，售电公司从简单价差套利转向综合能源服务。国家电网预测，2025年综合能源服务市场规模将达0.8-1.2万亿元，新型市场活力加速释放
