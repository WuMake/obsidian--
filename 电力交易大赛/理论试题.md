# 辅助服务市场
1. 调峰市场除了山西采用联合出清外，其他省份都是独立出请
2. 我国调频和备用辅助服务市场采用（与电能量市场出清、独立出请）模式
3. 对应不同的辅助服务调用方式，辅助服务的价格机制可以分为基于（统计成本）和基于（市场价格）。
4. 在电能量与调频联合出清中，参与调频服务的资源进行报价，报价具体包括（容量报价、里程报价、愿意提供的调频容量）部分。
5. 自动发电控制属于二次调频；一次调频是指当电力系统频率偏离额定值时，发电机组通过调速器的自动调节作用，增减发电机的出力，以维持系统频率稳定的过程。它是电力系统的一种自然响应机制，不需要人工干预，能够快速响应系统频率的变化。

# 容量市场
1. 容量成本与发电量无关
2. 容量成本回收机制大体分为：容量市场机制、稀缺定价机制、容量成本补偿机制
3. 容量市场是以（市场竞争）的方式形成容量价格实现发电容量成本回收，是竞争性电力市场的有机组成，主要适用于电能量市场发展已经相对完善的地区。

# 多选概念
1. 电力金融市场包含交易主体、交易品种、交易规则方面内容，没有交易时间
2. _垄断竞争市场_（Monopolistic competitive market）是指许多厂商生产相近，但不同质量的商品市场，是介于完全竞争和完全垄断的两个极端市场结构的中间状态。企业的生产规模较小，进入和退出市场比较容易
3. 电力交易方式有：双边、集中竞价、拍卖。没有协商交易
4. 萨利亨特在其《电力市场竞争》一书中从美国电网调度和电力交易的管理体制角度提出了几种电力市场模式，它们是（一体化、分散、集中）
5. 交易机构根据市场主体签订的交易合同及集中交易结果和执行结果，出具（电量电费、辅助服务费、输电服务 ）等结算依据。
6. 现货电能量市场中用户侧允许申报偏差外收益处理机制，说法正确的有（申报的用电需求曲线与其实际用电曲线之间的偏差率不得超出允许偏差范围；偏差率以小时为单位计算）
7. 下面哪些属于发电机组在日前市场报价要考虑的费用（启动、空载、电能）
8. 我国需求侧资源参与电力市场的方式——辅助服务、电力现货
9. 金融输电权即为获取阻塞盈余的权利。一般包括：注入节点、流出节点、功率、时间段等要素
10. 基于价格的激励政策主要包括固定“上网电价、溢价机制、招标电价、差价合约
11. 市场交易价格可以通过（自主协商、集中竞价）确定
12. 计算基本电价的方法有（ 综合成本、长期边际成本）
13. 电力市场引入竞争：怅惘分开，鼓励售电

4. （交易估计法和合计分摊法）根据每个输电转运服务用户的功率对整体系统所带来的影响来承担网损，这种方法最大限度地保证分摊的公平合理。
