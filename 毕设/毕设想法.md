# 风电预测
## 方向一：从数据处理、特征选择到预测模型
文章未局限于单一模型改进，而是针对风电预测中 “数据质量差、特征冗余多、经济关联性弱” 三大核心痛点，设计了四模块协同的 ARFEAT 框架（Anomaly Repair+Feature Enhancement+Asymmetric loss optimization+Transformer-based modeling），每个模块均有针对性创新：
1. **半监督学习驱动的数据异常修复：兼顾 “准确性” 与 “物理一致性”** 针对风机传感器误差、环境干扰导致的数据异常（约占数据总量 5%），提出 “KNN 异常检测 + LightGBM 数据插值” 的两步法：先用无监督 KNN 算法识别偏离风机功率 - 风速 S 型物理曲线的异常点（通过欧氏距离判断数据点与邻居的偏差），再用监督学习模型 LightGBM 基于风电物理公式插值修复，且对比单特征（仅风速）与多特征（风速 + 温度 + 湿度）插值效果，证明多特征方法能提升数据抗噪性，使修复后的数据更贴合实际物理规律。
2. **Fisher Score 特征筛选：高效解决 NWP 数据冗余问题** 面对 6 个来源、约 160 个数值天气预报（NWP）特征的冗余问题（易导致模型过拟合、训练耗时增加），采用 Fisher Score 筛选高判别力特征 —— 通过 “最大化类间距离、最小化类内距离”，最终选取 Top16 特征（以不同高度 / 方向的风速为主，如 925hPa 东西向风速、10m 高度南北向风速），既降低特征维度，又保留与风电输出强相关的关键信息，筛选结果与工程经验（风速是风电预测核心因子）高度契合。
3. **政策导向的非对称损失函数：首次将 “经济成本” 深度融入模型优化** 突破传统 MSE/MAE 等对称损失函数的局限，结合中国电力市场规则（高估风电需支付高额缺电罚款，低估仅产生少量弃电成本），设计非对称损失函数：对 “高估误差（\(P>Y\)）” 设置更大惩罚系数\(\beta\)（测试时\(\beta=0.017\)），对 “低估误差（\(P<Y\)）” 设置较小系数\(\alpha\)（测试时\(\alpha=0.01\)），直接将经济损失量化为模型优化目标，使预测结果更贴合风电场实际收益需求。
4. **Transformer 编码器 - 解码器架构：适配风电时序数据特性** 针对风电功率的长时序依赖性，采用支持并行计算的 Transformer 结构：编码器通过 “全连接层 + 位置编码 + 自注意力机制” 捕捉历史数据关联，解码器通过 “前向掩码” 确保预测不依赖未来数据，相比传统 RNN/GRU 等序列模型，能更高效地学习多时间步的风电变化趋势，且支持 24/48/96 步（对应 6/12/24 小时）的短期预测需求。
